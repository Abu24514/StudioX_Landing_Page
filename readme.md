# StudioX

StudioX is a modern, cinematic landing page for a video production and content studio.  
The website focuses on visual storytelling, smooth animations, and a premium brand experience.

## Overview

StudioX showcases how creativity and technology come together to build powerful brand stories.  
The design is clean, mobile-first, and image-driven, with a strong focus on motion and mood.

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- GSAP (animations)  
- Lenis (smooth scrolling)

## Features

- Mobile-first responsive layout  
- Smooth scrolling powered by Lenis  
- Scroll-based and entrance animations using GSAP

## Animations

- Hero section text animations on page load using GSAP  
- Scroll-triggered text reveal animations for Work, Brand, and Footer sections  
- Image overlay resize animation on scroll  
- Card animations with staggered motion on scroll  
- SVG arrow stroke animation with infinite loop  
- Smooth scrolling implemented using Lenis  
- Responsive navigation menu animation for mobile

3. No additional setup required
